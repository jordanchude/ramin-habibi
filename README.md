# Ramin Habibi Website

A Webflow-based website project for Ramin Habibi.

## Project Structure

```
ramin-habibi/
├── *.html                   # Main website pages
├── css/                     # Stylesheets
├── js/                      # JavaScript files
├── images/                  # Website images
├── fonts/                   # Font files
├── brochure/                # Brochure pages
├── gallery/                 # Gallery sections
├── registration/            # Registration flows
├── template/                # Template pages
├── assets/                  # Additional project assets
│   ├── images/             # Extra image files
│   ├── documents/          # PDFs, documents, etc.
│   └── archive/            # Archived/old files
├── docs/                   # Project documentation
└── README.md              # This file
```

## Getting Started

1. The main website files are at the root level for easy access
2. Additional assets are organized in the `assets/` directory
3. Documentation and notes can be added to the `docs/` directory

## Website Structure

The website includes:
- **Main Pages**: `index.html`, `about.html`, `news.html`, etc.
- **Gallery Sections**: Located in `gallery/` directory
- **Registration Flows**: Located in `registration/` directory  
- **Brochure Pages**: Located in `brochure/` directory
- **Template Pages**: Located in `template/` directory

## Assets

- **Website Images**: Located in `images/` directory (part of the website)
- **Additional Images**: Located in `assets/images/` with clean, descriptive naming
- **Documents**: Place any PDFs, brochures, or other documents in `assets/documents/`
- **Archive**: Use `assets/archive/` for old or unused files

## Development Notes

- This is a Webflow-exported website ready for customization
- Main website files are at the root level for easy editing
- Additional assets are organized separately for better management
- The structure supports future expansion and collaboration
